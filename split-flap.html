Split-flap display

An airport departure board, rendered in code

Design · March 2026

Split-flap displays, the kind you'd find in train stations and airports, use mechanical tiles that flip through characters to land on each letter. The satisfying clatter of the flaps cycling through was once the sound of departure.

This is a recreation built with Claude Code and Cursor. Each tile animates through a short random sequence before settling, approximating the staggered, column-by-column wave of a real board updating.
