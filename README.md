# jacquelinekoerner.github.io

Personal portfolio — writing, design work, and case studies.

Built with React (no framework, no build step). Hosted on GitHub Pages.

## Stack

- Vanilla HTML + React via CDN
- Cormorant Garamond + DM Mono
- No dependencies, no build tools — just open `index.html`

## To run locally

```bash
open index.html
```

## Structure

```
index.html        # Everything lives here
README.md         # You're reading it
```

## To update

- **About / bio** — edit the `about-text` section in `index.html`
- **Writing** — update the `data.writing` array with new Substack posts
- **Design** — the Figma link lives in `data.design`
- **Now** — update the `#now` section whenever life changes
